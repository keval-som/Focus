# Assets

Place your extension icons here:

- `icon16.png`  — 16×16 px  (used in the toolbar)
- `icon48.png`  — 48×48 px  (used on the Extensions page)
- `icon128.png` — 128×128 px (used in the Chrome Web Store)

You can generate quick placeholder icons at https://favicon.io or use any image editor.
Until real icons are added, Chrome will show a default grey puzzle-piece icon.
